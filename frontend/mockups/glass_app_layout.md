# Макет интерфейса Telegram WebApp

Ниже приведён упрощённый текстовый макет, иллюстрирующий ключевые блоки экрана и взаимное расположение элементов во вкладке «Отчёт».

```
┌─────────────────────────────────────────────┐
│  Шапка                                      │
│  ┌────┐  «СТРОЙИНВЕСТ»                      │
│  │Лого│  Ежедневные отчёты по объектам   ●  │
│  └────┘                                    │
├─────────────────────────────────────────────┤
│ ┌──────────── Табы ─────────────┐           │
│ │ Отчёт │ История │ Доступ      │           │
│ └───────────────────────────────┘           │
│                                             │
│ ┌──────────── Карточка отчёта ────────────┐ │
│ │  Заголовок + текущая дата              │ │
│ │ ┌────────────────────────────────────┐ │ │
│ │ │ Объект (иконка + select)           │ │ │
│ │ │ Вид работ (иконка + select)        │ │ │
│ │ └────────────────────────────────────┘ │ │
│ │ ┌─────┬─────┬─────┐                    │ │
│ │ │Дата │Объём│Техника│                   │ │
│ │ └─────┴─────┴─────┘                    │ │
│ │ Люди (поле ввода с иконкой)            │ │
│ │ Комментарий (textarea)                 │ │
│ │ Фото (дропзона + предпросмотр 3×)      │ │
│ │ ┌───────────────┬───────────────────┐ │ │
│ │ │ Кнопка «Отправить»│ Полоса прогресса │ │
│ │ └───────────────┴───────────────────┘ │ │
│ └────────────────────────────────────────┘ │
│                                             │
└─────────────────────────────────────────────┘
```

## Ключевые изменения

- Карточка отчёта располагается внутри стеклянного контейнера с радиальными подсветками по краям.
- Управляющие кнопки, инпуты и табы используют градиентные фоновые переходы и сглаженные углы.
- Внизу карточки предусмотрен прогресс-бар, который синхронизирован с процессом отправки файлов на бэкенд.

## Дополнительные вкладки

- Вкладка «История» повторяет сетку фильтров в верхней части, после которой следуют карточки предыдущих отчётов с миниатюрами фотографий.
- Во вкладке «Доступ» расположены поля поиска, выбора проекта и роли, а также список текущих назначений с кнопкой «Изменить» напротив каждой записи.

Макет можно использовать как ориентир при верстке или обсуждении дизайна с командой.
