# Макет интерфейса с закреплённой кнопкой «Закрыть»

```
╭────────────────────────────────────────────────────────╮
│ Telegram WebView Safe-Area                             │
│                                                        │
│  ╭─────────────────────────────── Fixed Header ───────╮│
│  │  ◄ Закрыть   •   Ежедневные отчёты   •   ИП       ││
│  │  ───────────────────────────────────────────────  ││
│  ╰───────────────────────────────────────────────────╯│
│                                                        │
│  ╭───────────── Стеклянная карточка ─────────────────╮ │
│  │ Ежедневный отчёт        12.11.2025                │ │
│  │ ───────────────────────────────────────────────── │ │
│  │  Объект   [ЖК «Северный» ▼]                       │ │
│  │  Вид работ [Бетонирование ▼]                      │ │
│  │                                                   │ │
│  │  Дата  ┌──────────────┐  Объём ┌───────┐ м³      │ │
│  │        │ 2025-11-12   │        │ 12,5  │         │ │
│  │  Техника ┌───┐ шт.     Люди ┌───────┐            │ │
│  │          │  3│          │  7    │                │ │
│  │                                                   │ │
│  │  Комментарий                                      │ │
│  │  ┌─────────────────────────────────────────────┐  │ │
│  │  │ Бетонирование ростверка...                  │  │ │
│  │  └─────────────────────────────────────────────┘  │ │
│  │                                                   │ │
│  │  Фото (до 3 превью)                               │ │
│  │  [ + ]  [ preview ]  [ preview ]                  │ │
│  │                                                   │ │
│  │  [ Отправить отчёт ] ────────▰▰▰▰▰─────── 82%     │ │
│  ╰───────────────────────────────────────────────────╯ │
│                                                        │
│  Tabs:  [Отчёт] [История] [Доступ]                     │
│                                                        │
╰────────────────────────────────────────────────────────╯
```

* Фиксированная панель сверху перекрывает жест «свайп вниз», повторяя интерфейс Telegram.
* Основные блоки повторяют стеклянные карточки из макета: прозрачные панели, закругления 32–44px.
* Прогресс-бар располагается под кнопкой отправки, а превью фото ограничены тремя слотами.
* Вкладки размещены внизу карточки и переключают содержимое внутри стеклянного контейнера.
